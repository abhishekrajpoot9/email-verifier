# Email verification module
This project is a backend-based email verification utility built using Node.js.  
It verifies whether an email address is properly formatted and checks if the mailbox may exist using SMTP validation.

## Features

- Email syntax validation
- DNS MX lookup
- SMTP mailbox verification
- Typo detection
- Jest test cases

## Tech Stack

- Node.js
- JavaScript
- SMTP Protocol
- DNS Module
- Jest
 
 ## Installation

```bash
npm install
```
 
 ## Run Project
 ```bash
node index.js
```
## Run Tests

```bash
npm test
```
